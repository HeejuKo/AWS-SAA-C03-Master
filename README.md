# ☁️ AWS SAA-C03 마스터 (Master) - 문제풀이 프로그램

AWS Certified Solutions Architect - Associate (SAA-C03) 시험 대비를 위한 **총 684문제** 완벽 복원 및 풀이 프로그램입니다. 
복잡한 설치 과정 없이 누구나 클릭 한 번으로 쾌적한 환경에서 문제를 풀고 오답 노트를 관리할 수 있습니다! 

---

## ✨ 주요 기능
* **완벽한 UI/UX:** 다크 모드 기반의 깔끔한 디자인으로 눈 피로도 최소화
* **스마트한 문제 풀이:** 단일 선택(라디오 버튼)과 다중 선택(체크박스) 완벽 자동 분리
* **자동 오답 노트:** 틀린 문제는 자동으로 기록되어 `❌ 오답 노트` 모드에서 복습 가능
* **다양한 학습 모드:**
  * `📝 실전 모의고사`: 실제 시험처럼 65문항씩 끊어서 풀기
  * `🔀 전체 랜덤 풀기`: 684문제를 무작위로 섞어서 풀기
  * `❌ 오답 노트`: 내가 틀렸던 문제만 모아서 다시 풀기

---

## 📥 초간단 다운로드 및 실행 방법

컴퓨터에 파이썬(Python)이 깔려있지 않아도 괜찮습니다. **알아서 다 설치해 줍니다!**

### 1단계: 다운로드 및 압축 풀기
1. 이 화면 우측 상단의 초록색 **`<> Code`** 버튼을 클릭합니다.
2. 메뉴 맨 아래의 **`Download ZIP`**을 클릭하여 파일을 다운로드합니다.
3. 다운로드한 `AWS-SAA-Master-main.zip` 파일의 **압축을 완전히 해제**합니다. (⚠️ 중요: 압축을 풀지 않고 ZIP 파일 안에서 바로 실행하면 오류가 발생합니다!)

### 2단계: 프로그램 실행 (Start.bat)
1. 압축을 푼 폴더 안으로 들어가서 ⚙️ **`Start.bat`** 파일을 더블클릭합니다.
2. **까만 창(터미널)이 뜨면서 아래 작업이 자동으로 진행됩니다:**
   * **파이썬이 없는 경우:** 자동으로 파이썬을 다운로드하고 설치합니다. (설치가 끝나면 창을 닫고 `Start.bat`을 한 번 더 실행해 주세요!)
   * **파이썬이 있는 경우:** 필요한 환경을 자동으로 세팅한 후, 1~2분 내로 인터넷 브라우저가 열리며 프로그램이 시작됩니다.

## 💡 사용 꿀팁 및 주의사항

* **내 학습 기록 보존하기:** 문제를 풀면 폴더 안에 `my_progress.json` 이라는 파일이 자동으로 생성됩니다. 이 파일이 바로 여러분의 **'오답 노트 및 정답 기록'**입니다. 이 파일을 지우면 학습 기록이 모두 날아가니 주의하세요! (기록을 초기화하고 싶다면 프로그램 좌측 메뉴의 `🗑️ 초기화` 버튼을 누르면 됩니다.)
* **프로그램 종료하기:** 공부를 다 하셨다면 열려있는 인터넷 브라우저 창을 닫고, 뒤에 켜져 있는 까만 창도 `X`를 눌러서 닫아주시면 완전히 종료됩니다.
* **PC 보안 경고가 뜰 경우:** `Start.bat` 실행 시 윈도우의 'PC 보호(SmartScreen)' 파란색 창이 뜰 수 있습니다. 당황하지 마시고 **`추가 정보`** 글씨를 누른 뒤 **`실행`** 버튼을 누르시면 정상적으로 켜집니다.
